Welcome to My Flask App

This Application is built as a cloud project on python flask framework. It usues algorithms like yolo and kamines and cosine similarity. This poject uses the image detection allgorithm yolo to identify and catagorise an image after matching it from a total of 601 different openCV classes. After that, we use kamines and cosine similarity algos to detect the color and to display with the image.
